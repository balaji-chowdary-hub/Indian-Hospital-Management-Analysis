# Indian-Hospital-Management-Analysis
Healthcare Analytics Dashboard built using Power BI, analyzing 100K+ hospital encounters from 2019–2023. Provides insights on revenue, patient demographics, LOS, and operational efficiency. Helps improve decision-making, resource utilization, and patient outcomes in hospital systems.


# 🏥 Healthcare Analytics Dashboard

## 📌 Project Overview
This project presents a comprehensive Healthcare Analytics Dashboard built using Power BI. It analyzes hospital data from 2019 to 2023, covering over 100,000 patient encounters and 79,000 patients.

The dashboard provides insights into hospital operations, financial performance, and patient demographics to support data-driven decision-making.

---

## 🎯 Objectives
- Analyze hospital performance and patient trends
- Monitor financial metrics and revenue growth
- Evaluate operational efficiency (LOS, occupancy, satisfaction)
- Identify improvement areas using data-driven insights

---

## 📊 Key Metrics
- Total Encounters: 100,847
- Total Patients: 79,231
- Departments: 40
- Doctors: 1,500

---

## 📈 Dashboard Features

### 🔹 Executive Summary
- KPI overview
- Top departments
- Revenue trends
- Demographics (age, gender, insurance, state)

### 🔹 Clinical Operations
- Length of Stay (LOS) analysis
- Emergency trends
- Diagnosis insights
- Occupancy vs satisfaction

### 🔹 Financial Overview
- Total revenue trends (Year & Month)
- Billing status (Paid, Pending, Overdue, Disputed)
- Outstanding balances by state
- YoY revenue growth

### 🔹 Performance Analysis
- Doctor performance
- Department efficiency
- Patient satisfaction scores

---

## 🧠 Key Insights
- Chronic patients cost 2.3× more than normal patients
- Readmission rates exceed the 20% target
- Top 3 states contribute ~60% of outstanding dues
- High occupancy reduces patient satisfaction

---

## 💡 Recommendations
- Implement post-discharge follow-ups
- Optimize billing recovery in high-outstanding states
- Improve ICU & Cardiology efficiency
- Expand capacity in high-occupancy departments
- Invest in chronic disease prevention programs
- Improve doctor-patient interaction training
